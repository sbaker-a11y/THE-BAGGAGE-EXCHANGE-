Experiences

Carry On Workshops.

Practice the work.

Book or inquire

Carry On Workshops by THE Baggage Exchange are guided experiences that help participants name what they carry, set down what no longer serves them, and leave with practical tools for moving forward with lighter bags.

Practice the work

Name it, set it down, carry tools forward.

Name what you carry

Guided experiences that help you see what you have been holding.

Set down what no longer serves

Practice letting go of the weight that is not yours to keep.

Leave with practical tools

Move forward with lighter bags and skills tied to the H.O.P.E. Curriculum.

The H.O.P.E. Curriculum

Your H.O.P.E. Bag™

What you carry home

Workshop attendees take home a working H.O.P.E. Bag™: everything in the brunch bag, plus a session workbook and a tool tied to the skill you practiced.

The H.O.P.E. Bag™ is earned in person, never sold. The H.O.P.E. Shop sells tools anyone can buy.
